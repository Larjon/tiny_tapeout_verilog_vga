## How it works

This project generates a VGA video signal for a 640x480 display.  
A small red square moves across the screen and bounces off the edges, similar to the classic DVD screensaver animation.

The VGA timing (HSYNC, VSYNC, pixel position) is generated by the `hvsync_generator` module.  
The main module calculates the position of the square and updates it once per video frame using the VSYNC signal.  
When the square reaches the edge of the visible screen area, its velocity is reversed, creating a bouncing effect.

The square is drawn by comparing the current pixel coordinates with the square's position.

## How to test

1. Connect the design to a VGA display using the TinyVGA PMOD pinout.
2. Provide a clock of approximately **25 MHz** (the project is configured for **25.2 MHz**).
3. Reset the design.
4. The monitor should display a **black screen with a red square bouncing around the screen**.

No user inputs are required for normal operation.

## External hardware

The project uses a **VGA display connected through a TinyVGA PMOD adapter**.

Required hardware:
- VGA monitor
- TinyVGA PMOD (or equivalent resistor ladder VGA interface)
- FPGA or Tiny Tapeout chip providing a ~25 MHz clock
